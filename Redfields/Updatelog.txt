**Alpha Release - v0.3.0.0**
Added sword upgrades, knockback, stab attacks, replayability, and more!

**[NEW] Upgrade Shop**
Added shop for upgrading equipment.
- Added four swords: rusty (starter) sword, iron sword, steel sword, gilded sword.
- Added totem to open the shop at touch.
- Added shop GUI.
- Added stats for upgrades when hovering over them.
- Added upgrade class.
- Fixed rfb#005: Game crashes when opening store from retrieving player data.
- Fixed rfb#006: Game crashes when opening store from unknown upgrades.
- Fixed rfb#007: Game will not start due to upgrade constructor.
- Fixed rfb#009: Purchasing upgrade subtracts more gold than what you have.

**[NEW] Knockback**
Added knockback effect to enemies and the player.
- Added knockback variable.
- Added force variable to produce knockback effect.
- Added weight variable to defend against knockback.
- Knockback is affected by speed.
- Knockback is afffected by critical hit.
- Knockback is divided when multiple enemies are hit.
- Player force is affected by level system.
- Enemy force is affected by game difficulty.
- Player weight is affected by level system.
- Enemy weight is affected by game difficulty.
- Fixed rfb#002: Knockback only affects player when the player is moving.
- Fixed rfb#003: Player weight is set as 3 on game start, but its base is 4 in the level system.

**[NEW] Slash / Stab Attack**
Added stabbing attack.
- Renamed normal attack to slash attack.
- Stab knockback is 1.5x stronger than slash.
- Stab damage is 0.75x as strong as slash.
- Stab cooldown is 1.5x longer than slash.

**[NEW] Game Restart**
Added ability to restart the game after dying.
- Added replay after player death by pressing spacebar.
- Game wil restart at wave 0.
- Player will keep their level/XP.

**[NEW] Enemy Nametags**
Added names above enemy healthbars.
- Added nametag displays.
- Adjusted font size based on character count (to fit on health bar).

**Player**
Minor adjustments and bug fixes to the player.
- Nerfed level system effects.
- Adjusted player sword positions.
- Added lastMove variable to character class to distinguish player moves.
- Fixed rfb#004: Player has no damage or cooldown.
- Fixed rfb#008: Player health resets to max when damaged.

**Enemies**
Minor buffs to enemy drops.
- Buffed coin drop chance multiplier from 50 to 60.
